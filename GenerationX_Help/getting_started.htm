Getting Started

The first time you launch GenX it will attempt to load a file called GenerationX_data.ged in your Documents folder. If that file does not exist, you are prompted to select a GECOM file to open. You can changed the default file location in the preferences. Alternatively you can cancel the open dialog and GenX will create a new empty database for you. Once you have opened a GEDCOM file or created a new database you are ready to begin working with your data.

GEDCOM files must end in .ged for GenX to open them.
