Merging GEDCOM files

You can merge a second GEDCOM file into your current database by selecting Merge from the File menu, and navigationg to the file you want to merge in the resulting open dialog.

GenX attempts to bring the records into your current database. If the file being merged contains any records with the same label as a record in your current database that are not identical to the records in the current database, you will be prompted to choose witch record to keep. The file being merged is not altered in any way.
